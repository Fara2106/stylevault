# StyleVault — Piano Fase A (solo UI)

> Esecuzione autonoma inline autorizzata dall'utente ("vai fino in fondo, fai tutto quello
> che devi"). Spec di riferimento: `docs/superpowers/specs/2026-07-07-stylevault-design.md`.

**Goal:** UI completa e funzionante con persistenza localStorage, meteo reale Open-Meteo,
avatar SVG, design system lusso "light editorial". Niente backend (Fase B).

**Stack:** React 19 + Vite (JSX), react-router-dom 7, i18next, Vitest, @fontsource
(Playfair Display + Inter). Nessuna altra dipendenza.
